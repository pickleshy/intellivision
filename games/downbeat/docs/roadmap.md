# DOWNBEAT - Roadmap

## Metronome / Cursor

- [ ] Move the metronome dot one space ahead of the current beat (lead cursor showing where the next hit will land)
- [ ] Consider not having the metronome sound on every beat, but only the active ones, so the player knows which beats to hit. Alternative: two distinct metronome sounds (louder tick on active beats, softer tick on passive/dash beats)
