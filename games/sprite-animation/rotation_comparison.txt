KEY DIFFERENCES:
================

X-AXIS ROTATION:
- Letter flips top-to-bottom
- Width stays same, height compresses
- Edge view is horizontal line
- Best for "flipping" effects

Y-AXIS ROTATION:  
- Letter turns left-to-right
- Height stays same, width compresses
- Edge view is vertical line
- Best for "spinning" or "turning" effects

VISUAL METAPHOR:
- X-axis = flipping a coin (heads/tails)
- Y-axis = spinning a coin (edge view)

FOR TITLE SEQUENCES:
Y-axis rotation is typically more dramatic and recognizable
for text effects like letters "spinning in" to form words.
